# Projeto 1 - Rede de cinema

## Sobre o projeto 📚:
Será criado e desenvolvido diversos diagramas para uma rede de cinemas que, por sua vez, deseja informatizar o processo de reserva de ingressos em suas diversas unidades, com o objetivo de um sistema que permita que clientes consultem filmes disponíveis em cartaz, escolham sessões, etc. Além do sistema devendo também permitir que administradores gerenciem filmes, sessões e salas em diferentes locais.
Laboratório 1 da disciplina Modelagem de Software Orientado a Objetos.
